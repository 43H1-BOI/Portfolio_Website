## How to Run

Simply open `index.html` in a web browser, or serve it using any web server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8080
```

Then navigate to `http://localhost:8080` in your browser.
